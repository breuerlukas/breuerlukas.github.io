PORTFOLIO

My biggest projects

Dulno
Kubernetes Flutter
Dulno is pursuing the idea of replacing analog paper stamp cards with digital stamp cards in the form of an app. I noticed that a digital stamp card has many advantages for both users and customers, as well as for the distributors of the stamp cards. The project is already being used by real customers and is helping to simplify the lives of many people, thereby creating real added value.

Taskwolf
Java Javascript
Taskwolf is a software automation platform that should enable everyone to quickly automate everyday processes without requiring extensive prior knowledge. To do this, workflows are created that have a trigger and execute actions when triggered. This simple principle makes it possible to automate a large number of tasks quickly.

The project had a wide range of supporting functions designed to assist with automation. Although I invested a lot of time in the project, it unfortunately never achieved any significant success in terms of a broad user base, which is why I discontinued development on the project.

Panel
Java Kubernetes
The panel is a project I developed to manage the infrastructure of my other projects efficiently and securely. Among other things, support functions were integrated into the panel to enable targeted responses to customer requests. The panel also features statistics and analysis functions.

One of the core functions of the panel is a comprehensive Kubernetes administration interface. There, pods can be managed in a targeted manner, backups can be administered, and errors that have occurred can be evaluated.

Wing
C++ Arduino
This project deals with the complete automation of remote-controlled aircraft. On the one hand, complex and sophisticated algorithms assist the pilot in flying and try to neutralize safety risks. On the other hand, these systems are used to fully automate the operation of these aircraft, making pilot intervention unnecessary.

In the process, functionalities of already existing systems are extended and improved. The project aims to enable anyone to control such an aircraft and to make it easy to automate processes that are currently operated manually.
